# 👨‍💼 Employee Performance Predictor

## 📌 Overview
This project uses Machine Learning to predict employee performance based on HR data such as experience, salary, training hours, and department. It helps HR teams make data-driven decisions.

---

## 🎯 Problem
Manual employee evaluation is often biased and inconsistent. This project automates performance prediction using data analytics.

---

## 🧠 Solution
A classification model is trained on synthetic HR data to categorize employees as:
- High Performer
- Medium Performer
- Low Performer

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook
